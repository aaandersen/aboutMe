# anders.adalberth.com

Personal portfolio and résumé site for **Anders Adalberth Andersen** — BSc in
Business Administration & IT (CBS) and Technology Specialist at Microsoft,
focused on developing and orchestrating AI agents.

## Tech stack

- Vite
- TypeScript
- React
- Tailwind CSS
- shadcn/ui

## Getting started

```sh
# Install dependencies
npm install

# Start the dev server (http://localhost:8080)
npm run dev

# Production build
npm run build
```

## Deployment

Deployed on Netlify. The production build in `dist/` is committed and served
directly, so run `npm run build` before committing changes.
